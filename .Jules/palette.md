## 2024-11-20 - DropdownMenuTrigger Accessibility and Semantics\n**Learning:** In shadcn/ui, `DropdownMenuTrigger` renders as a `<button>` element. Nesting interactive elements like `<span>` with layout classes or failing to add `aria-hidden="true"` to decorative icons inside these triggers creates invalid HTML (nested interactives) and confusing screen reader output. \n**Action:** Always apply layout classes (e.g., `inline-flex`) directly to the `DropdownMenuTrigger` rather than wrapping inner content in spans, and ensure all inner purely decorative SVG icons have `aria-hidden="true"` when the trigger itself provides the accessible name.
