## 2024-03-15 - React Render Performance Optimization
**Learning:** Found an expensive calculation where `JSON.stringify` was being used synchronously inside a `useMemo` dependency array (`debouncedFingerprint`), blocking the main UI thread while typing in complex form builders. Replacing the expensive `JSON.stringify` check with direct object reference equality (since `useDebouncedValue` returns the exact object reference from the `useMemo` of `draftPayload` after the debounce interval) removed the overhead completely.
**Action:** Avoid expensive synchronous operations like `JSON.stringify` inside render cycles or frequently running `useEffect`/`useMemo` hooks. Rely on React's reference equality or debounce/throttle complex state diffing.
